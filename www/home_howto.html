The graphical user interface GraphApp provides a convenient and intuitive way to create or modify directed weighted graphs and apply graphical multiple comparison procedures. There is no additional requirement for programming languages, R and Java, on your system.
The R package gMCP also offers a GUI to conveniently create and perform Bonferroni-based graphical test procedures. To this end, the user will be required to invoke in R the gMCP package and subsequently call the GUI.
If there is any comments on the graphical user interface, please contact us.
